# Galt Timers

PWA для управления таймерами кальянов в заведениях общепита. Помогает кальянщикам не пропускать момент готовности и своевременно менять уголь.

## Суть проекта

Приложение заменяет бумажные заметки и таймеры на телефоне: один экран показывает состояние всех столов в реальном времени. Каждый кальян проходит цикл **готовность → вынес за стол → замена угля**, и приложение отслеживает каждую стадию, напоминая о действиях звуком, вибрацией и push-уведомлениями.

**Целевая аудитория:** кальянщики, официанты, владельцы кальянных.

## Возможности

- **Таймер готовности** — 10 минут с момента добавления стола
- **Таймер замены угля** — 10, 15, 20 или 25 минут (настраивается)
- **Тестовый режим** — интервал 10 секунд для проверки логики
- **Уведомления** — звук, вибрация, браузерные push (при скрытой вкладке)
- **Отчёты** — история закрытых столов с количеством замен угля
- **Сохранение состояния** — данные в `localStorage`, работа офлайн

## Стек

| Слой | Технология |
|------|------------|
| UI | React 19, TypeScript |
| Сборка | Vite 7 |
| Стили | Tailwind CSS 4, shadcn/ui, Radix UI |
| Состояние | React Context (без внешних сторов) |
| Хранение | localStorage |

## Архитектура

- **AppContext** — единый источник правды: столы, настройки, отчёты, очередь уведомлений
- **Типизация** — строгие типы для `Table`, `Hookah`, `Timer`, `Settings`; статусы как union-типы
- **Таймеры** — `setInterval` 1 сек, пересчёт `remaining` на каждом тике; при `endTime <= now` переход в следующее состояние и добавление алерта
- **Декомпозиция** — `HookahCard` (карточка стола), `CircularTimer` (SVG-прогресс), модалки через Radix Dialog

## Запуск

```bash
npm install
npm run dev
```

Открыть `http://localhost:5173` (или хост из вывода Vite).

## Сборка и деплой

```bash
npm run build
```

Артефакты в `dist/`. Готово к деплою на Netlify, Vercel, GitHub Pages (SPA, base path по умолчанию `/`).
